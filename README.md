## Terraform GitHub Lab

This repository contains Terraform configuration files used to automate the creation of GitHub repositories.  

### Features

- Uses the GitHub provider to create repositories.
- Demonstrates variables, outputs, and resource management.

### Required env vars

```
export GITHUB_TOKEN=<YOUR_PERSONAL_ACCESS_TOKEN_(CLASSIC)>
```
